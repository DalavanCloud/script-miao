FONTLOG
MiaoUnicode fonts for the Miao (Pollard) script
===========================================

This file provides detailed information on the MiaoUnicode font software.
This information should be distributed along with the MiaoUnicode fonts and any derivative works.


Basic Font Information
----------------------

MiaoUnicode is a set of Unicode fonts for the Miao (Pollard) script.
Font sources are published in the repository and a smith open workflow is used for building, testing and releasing.
