FONTLOG
ShiShan fonts for the Miao (Pollard) script
===========================================

This file provides detailed information on the ShiShan font software.
This information should be distributed along with the ShiShan fonts and any derivative works.


Basic Font Information
----------------------

ShiShan is a set of Unicode fonts for the Miao (Pollard) script.
Font sources are published in the repository and a smith open workflow is used for building, testing and releasing.
